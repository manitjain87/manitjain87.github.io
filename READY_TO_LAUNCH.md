3. Name: `manitjain.github.io` (exactly this)
4. Public repository
5. Upload **entire folder**:

   - blog/ folder (with all 10 HTML files)
   - README.md, etc.

**Your site goes live at:** `https://manitjain.github.io` (wait 2 min)

### Step 2: Custom Domain (15 min)
1. Buy `manitjain.com` at namecheap.com ($10/year)
2. Go to Advanced DNS
3. Add 4 A Records + 1 CNAME (see DEPLOYMENT_GUIDE.md)
4. Go to GitHub → Settings → Pages
5. Enter custom domain: `manitjain.com`
6. Create file `CNAME` with content: `manitjain.com`
7. Enable HTTPS

**Wait 30 min → Site live at:** `https://manitjain.com`
